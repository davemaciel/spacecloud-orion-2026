# SPACECLOUD ORION — Portal e Painel Principal

**Data do último update:** 06/03/2026

## Visão Geral
O **SpaceCloud Orion** é o site/portal principal da SpaceCloud: landing, informações, páginas institucionais e assets públicos.

## O que este projeto faz
- **Site institucional** da SpaceCloud
- **Landing pages** de campanhas
- **Assets e páginas estáticas**
- **Manutenção e avisos** quando necessário

## Estrutura
- `dist/` contém o build final
- Fontes variam conforme setup da página

## Execução
```bash
npm install
npm run build
```
